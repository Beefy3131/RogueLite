Privacy Policy — RogueLite Survivors

Effective date: July 10, 2026

The short version

RogueLite Survivors collects no personal data. There are no accounts, no ads, no analytics, no tracking, and no data leaves your device.

What is stored, and where

Your game progress (unlocked characters and maps, gold, permanent upgrades, lifetime statistics, and audio settings) is saved in your browser's / device's local storage. This data stays on your device, is not transmitted anywhere, and is deleted if you clear the site's data or uninstall the app.

Offline play

The game is a Progressive Web App. On first load it caches its own files (code, art, audio) on your device so it can run offline. This cache contains only game files — nothing about you.

Third parties

The game is hosted on Vercel, whose servers may log standard technical request information (such as IP addresses) to deliver the site, as any web host does. The game itself sends no data to any third party.

Children

Since no data is collected from anyone, no data is collected from children.

Changes

If a future update changes any of the above (for example, adding optional purchases), this policy will be updated first and the effective date revised.
